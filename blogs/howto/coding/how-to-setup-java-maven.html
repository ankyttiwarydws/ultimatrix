JAVA & Maven Setup Steps

JAVA Setup Steps:-

1. Download and install - jdk-17_windows-x64_bin.exe

2. Download and install - jre-8u311-windows-x64.exe

3. Set JAVA_HOME as below -
JAVA_HOME = C:\Program Files\Java\jre1.8.0_311

4. Set Path as below -
PATH = C:\Program Files\Java\jre1.8.0_311\bin

Or
Set JAVA_HOME and Path using Command Prompt with below commands -
setx JAVA_HOME "C:\Program Files\Java\jre1.8.0_311"
setx PATH "%JAVA_HOME%\bin";

setx -m JAVA_HOME "C:\Program Files\Java\jre1.8.0_311"
setx -m PATH "%JAVA_HOME%\bin";

5. Once setup is complete, check java version and home in command prompt with below commands -

Command: java -version
Output:
java version "17.0.1" 2021-10-19 LTS
Java(TM) SE Runtime Environment (build 17.0.1+12-LTS-39)
Java HotSpot(TM) 64-Bit Server VM (build 17.0.1+12-LTS-39, mixed mode, sharing)

Command: echo %JAVA_HOME%
Output:
C:\Program Files\Java\jre1.8.0_311

Setup is complete.

Maven Setup Steps:-

unzip apache-maven-3.8.3-bin.zip

export PATH=/opt/apache-maven-3.8.3/bin:$PATH

mvn -v

MAVEN_HOME = D:\Soft\apache-maven-3.8.3-bin\apache-maven-3.8.3
Path = %MAVEN_HOME%\bin
